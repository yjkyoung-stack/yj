### 이미지 내 텍스트 사용 시 공통 주의사항

#### 텍스트 삽입

* 세이프존을 고려해 메시지가 잘리지 않도록 합니다.
* 이미지 전체에 배치할 수 있으나, 최대 3줄까지만 사용합니다.

#### 텍스트 정렬

* 좌측 또는 중앙 정렬을 사용합니다.
* RTL 언어는 우측 정렬을 허용합니다.
* 한 포스트 내에서는 정렬 방식을 혼용하지 않습니다.

#### 텍스트 구성

* Headline과 Sub-headline 조합을 사용할 수 있습니다.
* Sub-headline은 선택적으로 구성합니다.
